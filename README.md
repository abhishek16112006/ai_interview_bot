# 🤖 AI Interview Bot

An AI-powered Interview Bot that helps users practice technical and HR interview questions. It generates intelligent responses and simulates real interview scenarios.

## 🚀 Features

- 🎤 Real-time Interview Simulation
- 🤖 AI-generated Questions & Answers
- 📊 Performance Feedback (optional)

- 🧠 Supports multiple domains (DSA, HR, etc.)
